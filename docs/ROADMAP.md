# Roadmap

## P0

- Publish signed artifacts to Maven Central.
- Add coroutine-friendly async APIs.
- Add Android-safe companion documentation that routes secret operations through a backend.

## P1

- Add typed models for subscriptions, POS terminals, products, and shop APIs.
- Add retry/backoff hooks that callers can opt into.
- Add generated API coverage from the MakePay OpenAPI spec.

## P2

- Add Dokka-generated API documentation.
- Add Kotlin Multiplatform feasibility notes.
- Add sample Ktor and Spring Boot integrations.
